Beautiful API - Implementação de API em Go

## Tecnologias e Ferramentas

- Linguagem Go (v1.19)

## Como usar

Este projeto utiliza `Makefile`. Dessa forma, ele foi configurado para facilitar a utilização deste projeto. A seguir estão alguns comandos:

Iniciar API:

```sh
make run
```

Buildar Projeto:

```sh
make build
```
